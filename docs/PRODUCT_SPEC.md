# AI 美髮染髮配方計算 App - Product Spec

## 定位

本產品是給美髮設計師使用的染髮配方輔助工具，不是自動決定染劑配方的 AI。公式與比例由本地規則引擎計算；品牌規則、資料來源與驗證狀態都以 JSON seed data 管理。

## 第一版功能

- 使用者輸入服務類型、品牌、產品線、目前色度、目標色度與目標色系。
- 使用者輸入漂髮史、黑染殘留、人工色素殘留、受損程度、髮量、髮長與白髮比例。
- 使用者確認是否需要白髮覆蓋、補色／中和、靠近頭皮操作與高風險提醒。
- 必須保留人工確認目前底色的流程。
- 輸出建議配方、雙氧、混合比例、施工流程、風險提醒、信心等級與來源摘要。

## 限制

- 未 verified 品牌規則只提供通用參考方向，不輸出精確克數。
- 補色／調彩克數不使用通用公式硬算；沒有品牌明確規則時只輸出方向性建議。
- 色卡照片與頭髮照片只能作為參考，不能取代現場判斷。
- 所有輸出都需要專業設計師確認。
