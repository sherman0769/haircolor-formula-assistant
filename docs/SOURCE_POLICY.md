# Source Policy

## 驗證狀態

- `verified`: 官方品牌網站、官方 PDF、官方技術手冊、官方 leaflet 或官方色卡可直接支持該產品線核心規則。
- `partial`: 來源可信但不完整，例如供應商代管技術手冊、舊版手冊、只涵蓋部分產品變體或區域不明。
- `unverified`: 通用教育資料、內部假設、部落格、非官方影片或缺乏可追溯來源的資料。

## 規則

- 沒有官方或可信來源的品牌規則不能標示為 `verified`。
- 任何 `partial` 或 `unverified` 規則不得輸出精確品牌配方克數。
- 所有 brand rule 必須包含 source metadata。
- 來源更新後需同步更新 `retrievedAt` 與 `sourceNote`。

## 補色資料

補色方向可由色彩學規則提示，但補色克數不得用通用公式硬算。除非品牌官方資料明確定義調彩濃度與加入比例，否則只能輸出方向性建議。
